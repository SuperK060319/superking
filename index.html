SuperKing CC 安装小课 00 / 08

给微信里的朋友

今晚把 CC 装好，明天就让它替你做第一个网页。

CC 就是 Claude Code。不会写代码也没关系，这份手册只要求你会复制、粘贴、回车。

页面不依赖国外字体、CDN 或图片，发到微信也能正常打开。

STEP 1 / 8

先确认你有一个终端

Windows 按 Win + R，输入 powershell 回车。Mac 打开「启动台」，搜索 终端。

后面的命令都在这里输入。看到黑色或白色命令窗口就算过关。

STEP 2 / 8

检查 Node.js

复制下面这行到终端。如果出现类似 v20.11.0 的数字，直接下一步。

$ node --version

提示找不到命令？先装 Node.js LTS。官网打不开时，用国内镜像下载 LTS 安装包。

Node.js 官网 国内镜像

STEP 3 / 8

安装 Claude Code

优先用系统自带安装方式。访问海外慢时，改用 npm 路线，配国内镜像会稳一点。

Windows $ winget install Anthropic.ClaudeCode
Mac $ brew install --cask claude-code

没有 brew 的 Mac 用户，先去 brew.sh 安装 Homebrew；Windows 找不到 winget，先更新微软商店里的「应用安装程序」。

STEP 4 / 8

如果上一步卡住，就走 npm 备用路线

这条路线更适合国内网络。先切 npm 镜像，再安装 CC。

1 $ npm config set registry https://registry.npmmirror.com
2 $ npm install -g @anthropic-ai/claude-code

安装后输入 claude --version。能看到版本号，就说明 CC 已经在电脑里了。

STEP 5 / 8

准备一把 API 钥匙

如果你能直接使用 Claude 官方账号，就按官方登录走。国内网络不稳时，可以先用 DeepSeek 作为模型供应商。

打开 DeepSeek 开放平台，注册，进入 API Keys，创建一把新钥匙。复制后马上保存，因为它通常只完整显示一次。

DeepSeek 开放平台

建议先充值少量金额测试。不要把 API Key 发到群里，也不要截图给别人。

STEP 6 / 8

把模型切到可用的线路

如果你用 DeepSeek，推荐装一个模型切换工具，比如 CC Switch。它的作用很简单：把 CC 的请求转到你配置好的供应商。

Mac $ brew tap farion1231/ccswitch && brew install --cask cc-switch

Windows 用户下载 .msi 安装包。GitHub 打不开时，找发你这份教程的人要国内网盘包。

CC Switch Releases

STEP 7 / 8

启动 CC

在桌面新建一个文件夹，叫 my-first-site。终端输入 cd 加一个空格，把文件夹拖进终端，再回车。

$ claude

第一次启动遇到选择项，先按默认选。进入对话界面后，输入 /model，确认使用你刚才配置好的模型。

STEP 8 / 8

让它做第一个作品

把下面这句话粘进去。它会生成一个可以双击打开的网页文件。

» 帮我做一个自我介绍网页，用单个 HTML 文件完成，页面要适合发到微信里打开。做完告诉我文件在哪里。

等它写完，打开文件夹，双击生成的 .html。看到页面，就算真正完成。

收工

你不是在学一个软件，你是在给自己装一个会干活的同事。

下一步，把你平时重复做的事情写成一句话交给它：改文案、做网页、整理表格、写脚本。先让它做小事，越用越准。
